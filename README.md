# CCAPDEV-MCO

### Where to access
```
https://archerlabs-ooc8.onrender.com
```

### How to initialize
```
npm init -y
```

### Packages Needed
```
npm install express mongoose cors jsonwebtoken multer bcrypt cookie-parser dotenv
```
```
npm install nodemon --save-dev
```

### How to run
```
npm run dev
```

### How to populate database
```
node src\scripts\seed_db.js
```
